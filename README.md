# Text QR Generator - version 'Novae'
### Forked from original ['python-gui_text_qr_generator'](https://github.com/KDTal1/python-gui_text_qr_generator) by KDTal1

## Features:
- Compilation:
    * Compiles entry text.
    * Compiles text files (word limit around 500, and .txt)
- QR Modes:
    * Binary
    * Consochain - Turns your text into a consonant chain
    * Reverse - Reverses your text.
    * Effigy - Effigy.
